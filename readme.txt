Navy Wars es un juego basado en el clasico hundir la flota para tu Nintendo DS.

Es el primer juego de nuestro equipo programado integramente usando nuestras librerias, NightFox’s Lib, y fue galardonado con el tercer premio en el Scenery Beta 2009 Homebrew International Showcase.
El objetivo del juego es simple, hundir todos los barcos enemigos antes que tu oponente hunda los tuyos.
Hemos optado por la reglamentacion mas basica, un disparo por turno, sin opcion a seguir disparando si aciertas. Cada jugador tiene un total de 6 navios, 1 de 4 casillas, 2 de 3 casillas y 3 de 2 casillas.
Una vez colocados todos tus navios en el tablero, en la posicion y direccion que quieras, empieza la partida.
Para disparar, simplemente tocaremos la casilla donde creemos que se esconde un barco enemigo. Si fallamos, veremos el disparo al agua, en caso de acertar, veremos la explosion correspondiente. El navio enemigo no sera visible completamente haste que este sea destruido (hundido).

Controles – Menu.
Puedes seleccionar las opciones del menu con el stylus o con el D-Pad, usando el boton A para aceptar.

Controles – Despliegue.
Para colocar los navios en el tablero, simplemente dibujalos con el stylus; para cambiar de tipo de navio, usa las teclas R y L. Puedes mover el tablero usando el D-Pad si eres diestro, o los botones A, B, X e Y si eres zurdo. Una vez colocados todos los navios, empezara la batalla.

Controles – Batalla.
Durante tu turno, usa el stylus para realizar el disparo, el D-Pad o los botones A, B, X e Y para moverte por el campo de batalla. Si mantienes pulsados los botones R o L, moveras la pantalla superior, que es donde esta colocada tu flota.
Puedes pulsar el boton SELECT en cualquier momento para rendirte y volver al menu.

Instalacion.
Para poder jugar a Navy Wars en tu Nintendo DS necesitaras disponer de una flashcard, si no dispones de ninguna, puedes jugar tambien en un emulador de DS para PC, como el no$gba.
Para instalar el juego, simplemente copia el archivo .nds en tu flashard y ejecutalo.
El juego usa el sistema NitroFS como sistema de archivos, si tu flashcard no soporta la funcionalidad de «Argv», deberas ejecutarlo usando el Homebrew Menu de Drunken Coders.

Esta ultima version esta recompilada, con lo que ya funciona en la CycloDS Evo I en modo DSi.
